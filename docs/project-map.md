# Карта проекта

## UI (экраны)

HomeScreen.kt — главный экран приложения

LoginScreen.kt — экран входа

RegisterScreen.kt — экран регистрации

MatchesScreen.kt — поиск учебных партнёров

PartnerProfileScreen.kt — профиль учебного партнёра

ChatScreen.kt — чат между пользователями

ProfileScreen.kt — профиль пользователя

EditProfileScreen.kt — редактирование профиля

## Навигация

AppNavigation.kt — маршруты и переходы между экранами

MainScreenWithBottomBar.kt — нижняя навигация приложения

## Данные

MockDataProvider.kt — тестовые данные пользователей

ChatRepository.kt — хранение и отправка сообщений

## Модели

User.kt — модель пользователя

ChatMessage.kt — модель сообщения

## Точка входа

MainActivity.kt — запуск приложения
